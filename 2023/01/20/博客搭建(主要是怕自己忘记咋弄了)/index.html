博客搭建(主要是怕自己忘记咋弄了)

网站本地启动

cd hexo-blog

hexo g

hexo server

博客hexo根目录在D:/blog

先修改完博文内容

👇更新本地hexo配置

先hexo g

再hexo d

配置完后准备发布到GitHub👇

git config –global http.sslVerify “true”

👆个人解决方案

按照网上的false并不能使用，更改为true后惊讶地发现可以使用

另外还要到设置中修改域名（每次博客更新都需要修改域名，每次更新设置的域名都会重置）
